# Object File Naming
## Description
- The file name must start with the Object Name without spaces and prefix
- The file name must end with the Object Type in Pascal Case
- There must be a dot (.) between the Object Name without spaces and prefix and the Object Type in Pascal Case
- The file name must have the extension .al

## ❌ Bad naming
Customer.PageExt.al, salesHeader.TableExt.al, Inventory Adjustment.Codeunit.al, NewTable.al
## ✅ Good naming
CustomerList.PageExt.al, SalesHeader.TableExt.al, InventoryAdjustment.Codeunit.al, NewTable.Table.al
